## 🚀 Objetivo

Facilitar o aprendizado de **PostgreSQL** por meio de exemplos práticos, exercícios e cenários reais, permitindo o domínio de consultas, modelagem e administração de bancos de dados relacionais.

---

## 📚 Sobre o Repositório

Este repositório é dedicado ao estudo e prática de banco de dados utilizando PostgreSQL, abordando desde fundamentos até recursos avançados.

Aqui você encontrará:

- Consultas SQL básicas e avançadas  
- Manipulação de dados (CRUD)  
- Modelagem relacional  
- Funções e procedures  
- Otimização de consultas  
- Exercícios práticos  

---

## 🧠 Tecnologias e Conceitos Abordados

Para utilizar este repositório:

- PostgreSQL instalado  
- pgAdmin ou DBeaver (opcional)  
- Conhecimento básico de lógica  

---

## 🔧 Como Utilizar

1. Clone o repositório:

```bash
git clone https://github.com/CharlesFreitas-DEV/SQL.git
```

2. Acesse o diretório:

```bash
cd SQL
```

3. Execute os scripts no PostgreSQL:

```bash
psql -U seu_usuario -d seu_banco -f arquivo.sql
```

4. Siga a ordem das pastas para evolução do aprendizado
